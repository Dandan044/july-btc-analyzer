## 文件结构

```

├── data/           # 原始 JSON 数据（当天覆盖）
├── logs/           # 执行日志（追加）
├── reports/        # 完整报告（独立文件）
```

## 任务路由

当收到任务指令时，读取对应的任务规则文件并严格执行：

| 任务 | 规则文件 |
|------|---------|
| 执行日报任务 | `tasks/daily-report.md` |
| 设定市场警报 | `tasks/set-alert.md` |
| 警报调试报告 | `tasks/alert-debug.md` |
| 即时分析任务 | `tasks/instant-analysis.md` |

### 触发方式

- **日报任务**：定时触发（9:00/21:00 GMT+8）
- **设定市场警报**：收到"设定市场警报"指令
- **警报调试报告**：收到"警报调试报告"指令
- **即时分析任务**：警报触发时自动调用
